# DESIGN.md

## 项目与用户画像
- 产品名：豆包工作室 Doubao Studio Desktop
- 用户群：AI 内容创作者，多账号管理需求，追求效率的 power user
- 设备：Windows 桌面端（Electron）

## 品牌与视觉方向
- **气质与意象**：深夜工作室，暗色环境里唯一的光源是屏幕——紫色氛围灯勾勒出桌面轮廓，专注、专业、有创造力。
- **视觉策略**：深色基底 + 高饱和紫色强调色，减少视觉噪音，让用户聚焦于内容创作本身。

## Design Tokens

### 色彩
- 底色：#0f0f14（接近纯黑，带微弱紫底）
- 二级底色：#1a1a24
- 三级底色：#24243a
- 表面色：#1e1e2e / hover #2a2a3e / active #32324a
- 边框色：#2a2a3e / light #38385a
- 强调色：主紫 #6c5ce7 / 浅紫 #a29bfe / 深紫 #5a4bd1
- 文本色：主 #e8e8f0 / 次要 #9898b8 / 弱化 #686888
- 状态色：空闲绿 #4ade80 / 忙碌琥珀 #fbbf24 / 异常红 #f87171

### 字体
- 字体族：Inter（英文/数字） + PingFang SC（中文） + Microsoft YaHei（Windows 中文回退）
- 等宽：JetBrains Mono / Fira Code（任务输入区）
- 节奏：标题紧凑 tracking-tight，正文宽松 leading-relaxed

### 圆角
- 标准：8px（卡片、按钮、输入框）
- 大圆角：12px（弹窗）
- 标签：全圆角 9999px

### 阴影
- 标准：0 2px 8px rgba(0, 0, 0, 0.4)
- 深层：0 8px 32px rgba(0, 0, 0, 0.6)

## 布局与响应式
- 三栏式：左侧面板（默认 340px，可折叠，拖拽调整 280-500px）+ 分隔线 + 右侧浏览器
- 左侧面板内部上下分区：账号列表（默认 45%）+ 任务控制台（55%），可拖拽调整比例

## 组件规范
- 按钮：ghost 图标按钮（8×8 圆角，hover 变色）/ accent 强调按钮（紫色实心）
- 状态指示点：小圆点 8px，绿/琥珀/红色
- 任务标签：全圆角药丸，小号字体，半透明底色
- 滚动条：6px 宽，深色圆角

## 动效与交互
- 面板折叠：transition-all duration-300
- hover 状态：200ms transition
- 加载覆盖层：backdrop-blur-sm + 半透明背景
- 鼠标拖拽分隔线时高亮紫色

## 设计禁忌
- 不要使用亮色/白色背景
- 不要使用大圆角卡片（超过 12px）
- 不要使用蓝紫色渐变（只用纯色紫）
- 不要过多装饰性元素，保持克制
